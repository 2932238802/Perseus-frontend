## 项目总览

### 技术栈

| 类别 | 技术 |
|------|------|
| 语言 | C++20 |
| 框架 | Qt 6 (Widgets, Network, Concurrent, WebEngineWidgets, WebChannel) |
| 构建 | CMake ≥ 3.18 |
| 平台 | Ubuntu / WSL2 |
| LSP | clangd (C/C++), rust-analyzer (Rust), neocmakelsp (CMake), pyright (Python) |
| 终端 | xterm.js + QWebChannel |
